🚀 部署步骤

Edge 浏览器安装步骤:

  1. 打开 Edge 浏览器，访问 edge://extensions/
  2. 开启"开发人员模式"（右上角开关）
  3. 点击"加载解压缩的扩展"
  4. 选择 build/cxmooc-tools 文件夹
  5. 扩展将自动加载并显示在工具栏

Chrome 浏览器安装步骤:

  1. 打开 Chrome 浏览器，访问 chrome://extensions/
  2. 开启"开发者模式"（右上角开关）
  3. 点击"加载已解压的扩展程序"
  4. 选择 build/cxmooc-tools 文件夹
  5. 扩展将自动加载并显示在工具栏
